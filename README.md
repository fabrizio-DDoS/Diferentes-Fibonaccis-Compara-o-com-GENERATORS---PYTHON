# Diferentes-Fibonaccis-Compara-o-com-GENERATORS---PYTHON
Comparação temporal de código utilizando Generators, OOP e Decorators no Python
Nesse experimento comparo um código limpo de geração de números conforme Fibonacci, e um Generator com mesma função.
Por mais que o Generator seja relativamente mais lento, há um desgaste muito menor da maquina e o processamento consome muito menos memória, já que o processo ocorre número após número.

Com uma diferênça de aproximadamente 2,175% o generator é mais lento.
